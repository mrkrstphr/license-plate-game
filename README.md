# 🚗 License Plate Game

> *Spot 'em all. Every state. Every province. Every road trip.*

**[▶ Play now →](https://mrkrstphr.github.io/license-plate-game/)**

The classic road trip game, now in your pocket. Tap a plate when you spot it, watch your progress fill up, and see if you can collect all 50 states + D.C. and every Canadian province before you hit the driveway.

---

## Features

- 🇺🇸 All 51 US plates — 50 states + Washington D.C.
- 🍁 All 13 Canadian provinces and territories
- 🗂️ Multiple named games — run one per trip, or per year
- 📅 Retroactive entry — log plates from a trip you already took
- 🔍 Search by state/province name or abbreviation
- 📊 Live progress bars for US and Canada separately
- 💾 Everything saves locally in your browser, no account needed
- 📱 Mobile-first, works great as a home screen app

## Tech Stack

- [React Router v7](https://reactrouter.com/) (SPA mode)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- Deployed to GitHub Pages via GitHub Actions

## Development

```bash
npm install
npm run dev
```

App runs at `http://localhost:5173`.

```bash
npm run build   # production build → build/client/
```

Deployment to GitHub Pages happens automatically on every push to `main`.
